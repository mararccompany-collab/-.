### 3️⃣ **المنتخبات الكاملة** 🏳️

  - الترتيب العالمي (FIFA Rank)
  - اللاعبون الأساسيون
  - المدرب والتشكيلة
  - الألقاب والمشاركات السابقة
  - نسب التوقعات الفردية

### 4️⃣ **الفرق والتشكيلات** ⚽

- تشكيلات لاعبيها
