### 🏳️ **المنتخبات** (الصفحة الجديدة الثانية)

**ماذا تجد هنا:**
- 12 مجموعة (A-L)
- 48 منتخب مع جميع البيانات
- أفضل 10 فرق
- معلومات مفصلة عن كل فريق

**الخطوات:**
1. انقر على "🏳️ المنتخبات"
2. اضغط على أي مجموعة لتوسيعها
3. اختر أي فريق لعرض التفاصيل
4. لاحظ أفضل 10 فرق في الأسفل

**معلومات الفريق:**
```
اسم الفريق       → بالعربية والإنجليزية
المجموعة        → A, B, C, ... L
الترتيب العالمي  → 1-50+
المدرب          → اسم المدرب
التشكيلة        → نظام اللعب (3-5-2، 4-3-3، إلخ)
اللاعبون        → أفضل 5 لاعبين
القوة           → مقياس 1-100
```

### 🎯 **الأدوار** (الصفحة الجديدة الثالثة)

**ماذا تجد هنا:**
- شرح كل مرحلة من البطولة
- احتمالات الخروج والتقدم
- معادلات توضيحية
- حقائق مهمة

**المراحل:**
```
1. دور المجموعات   → 48 فريق → 24 فريق
2. 1/16 النهائي   → 32 فريق → 16 فريق
3. ربع النهائي    → 16 فريق → 8 فريق
4. نصف النهائي    → 8 فريق → 4 فرق
5. مباراة البرونزية → 2 فريق → 1 فريق
6. النهائي        → 2 فريق → 1 بطل
```

**الاحتمالات حسب القوة:**
```
الفرق القوية جداً (FIFA 1-10):
- خروج المجموعات: 5%
- نصف النهائي: 40%
- الفوز: 50%

الفرق الضعيفة (FIFA 45+):
- خروج المجموعات: 70%
- نصف النهائي: 0%
- الفوز: 0%
```

### 🔮 **التوقعات** (الصفحة الجديدة الخامسة)

**ماذا تجد هنا:**
- 5 أنواع توقعات مختلفة
- أفضل 10 فرق لكل توقع
- نسب مئوية دقيقة
- شرائط بصرية توضيحية

**أنواع التوقعات:**
```
1. 📊 العبور من المجموعات
   مثال: البرازيل 95%

2. 🥇 قيادة المجموعة
   مثال: ألمانيا 85%

3. 🟢 الوصول نصف النهائي
   مثال: الأرجنتين 80%

4. 🟦 الوصول للنهائي
   مثال: فرنسا 75%

5. 🏆 الفوز بالبطولة
   مثال: إيطاليا 60%
```

**كيفية الاستخدام:**
1. اختر نوع التوقع من القائمة
2. انظر إلى أفضل 10 فرق
3. لاحظ النسب المئوية
4. قارن بين الفرق

## 🎮 نصائح الاستخدام

### ✨ نصائح مفيدة:

1. **استكشف جميع الصفحات**
   - كل صفحة تحتوي على معلومات فريدة
   - لا تتردد في التنقل بسهولة

2. **قارن بين الفرق**
   - استخدم صفحة المنتخبات للمقارنة
   - انظر إلى الترتيب والقوة

3. **اقرأ التوقعات بعناية**
   - التوقعات مبنية على بيانات حقيقية
   - تستند إلى الترتيب العالمي والتاريخ

4. **تابع التحديثات**
   - البيانات تحدّث دورياً
   - تحقق من الصفحة الرئيسية للأخبار

5. **شارك مع الأصدقاء**
   - شارك روابط الصفحات
   - شارك التوقعات والمعلومات

## 🔧 استكشاف الأخطاء

### إذا لم يحمل الموقع:

1. **تحقق من الخادم:**
   ```bash
   npm run dev
   ```

2. **حاول تحديث الصفحة:**
   - اضغط Ctrl+R (أو Cmd+R على Mac)
   - أو F5

3. **امسح ذاكرة التخزين المؤقت:**
   - Ctrl+Shift+Delete
   - اختر "كل الوقت"

4. **جرّب متصفح آخر:**
   - Chrome, Firefox, Edge

### إذا حدثت مشاكل في الواجهة:

1. **زيادة حجم النص:**
   - Ctrl + Plus (+)

2. **تصغير حجم النص:**
   - Ctrl + Minus (-)

3. **إعادة تعيين الحجم:**
   - Ctrl + 0 (صفر)
